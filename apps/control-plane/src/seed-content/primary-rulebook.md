# Ashen Gate Core Rulebook

This is the primary rulebook for the Ashen Gate demo table.

## Table Principles

- Keep turns short and consequential.
- Let player intent drive the scene.
- Ask clarifying questions only when the fiction is genuinely ambiguous.
- Favor momentum over exhaustive rules recitation.

## Core Resolution

- When an outcome is uncertain, interpret the player intent, the risk, and the immediate fiction first.
- Use the current session context and active sources to ground rulings.
- If a rule is missing, make a consistent ruling and mark it as an inference.

## GM Guidance

- Start scenes in motion.
- Name concrete sensory details.
- Give each NPC a motive, a pressure point, and a voice.
- End scenes with a choice, a complication, or a reveal.

## Safety and Tone

- Keep the table collaborative.
- Never overwrite player agency.
- Make the tone vivid, fast-moving, and easy to speak aloud.
