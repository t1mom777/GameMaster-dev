# Ashen Gate Table Reference

This supporting book gives the GM a quick reference for the first demo session.

## Quick Adjudication Notes

- If the table asks for a quick ruling, give the ruling first and the rationale second.
- Prefer one mechanical idea at a time.
- Offer a clear consequence instead of a long explanation.

## Session Flow

1. Open with a strong scene image.
2. Confirm what each player wants.
3. Add one pressure or threat.
4. Resolve with the smallest useful mechanic.
5. Leave the table with a reason to continue.

## Voice Runtime Hints

- Keep responses concise enough to speak naturally.
- When rule text matters, quote only the relevant fragment.
- When sourcing from the books, answer from the active materials first.

## Demo Table Tone

- Gothic, pressured, and readable.
- Clear enough for a first public session.
- Dramatic, but not verbose.
