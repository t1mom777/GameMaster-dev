# Kickstarter Outreach Calendar

Use this as the 6-8 week execution calendar for GameMaster Founder Access.

## Week 1: Foundation

- Create the Kickstarter draft.
- Publish `/kickstarter` on `game.dima.click` after deployment.
- Prepare the campaign hero image and 60-90 second video script.
- Add reward tiers and risks from `docs/kickstarter-campaign.md`.
- Submit the Kickstarter draft for review.
- When the prelaunch URL is approved, set `NEXT_PUBLIC_KICKSTARTER_PRELAUNCH_URL` and redeploy.

Primary message:

> Founder Access is opening for GameMaster: a voice-first virtual GM for real tabletop groups using one shared device.

## Week 2: Proof

- Record a demo showing sign in, upload rulebook, name players, start voice, and receive a spoken GM response.
- Record one real-table clip if possible.
- Prepare 5 short clips:
  - homepage
  - rulebook upload/indexing
  - ready-to-play page
  - voice start
  - GM answer
- Publish the builder post: `Why I’m building a virtual GM for one shared device`.

## Weeks 3-4: Warm Audience

- Post 3 times per week on X and LinkedIn.
- Share in approved TTRPG and AI-builder communities.
- DM 50 specific people:
  - indie TTRPG creators
  - actual-play hosts
  - GM tool reviewers
  - AI voice builders
  - small Discord/community owners
- Ask for feedback first, not pledges.

Short DM:

> I’m launching Founder Access for GameMaster, a voice-first virtual GM for real tabletop groups using one shared phone or laptop. Founder tiers include advanced model and voice configuration, including bring-your-own compatible providers where available. Would you be open to giving feedback on the prelaunch page before it goes live?

## Weeks 5-6: Conversion

- Announce the Kickstarter launch date.
- Ask warm contacts to follow the prelaunch page.
- Run a table test week with 5-10 real users.
- Collect 3 short testimonials.
- Add the strongest quote to the campaign page.
- Publish countdown posts 7 days, 3 days, and 1 day before launch.

## Launch Week

- Message warm backers directly when the campaign opens.
- Post launch copy on X, LinkedIn, Product Hunt updates, Indie Hackers, and approved TTRPG communities.
- Send backer updates on:
  - day 1
  - day 3
  - day 7
  - midpoint
  - final 72 hours
  - final 24 hours
- Keep pushing the CA$39 and CA$79 tiers as the primary choices.

## Weekly Scorecard

Track every Friday:

- Kickstarter followers
- backup email leads
- warm replies
- committed day-one backers
- playtesters booked
- testimonials collected
- demo clips recorded
- Android voice test status
- rulebook upload/indexing status
- founder model/voice configuration status
